# lotool

ただくじ引きするだけのソフト。
PowershellでGUIソフトを書く練習サンプルです。


# 
Windows10
.Net Framework v4以降
Powershell 5以降

# How to Use
・左側のテキストボックスに抽選したいものを改行区切りで入力
・抽選ボタンを押すと左側リストから一つ選ばれて右側に出力
・ログも見れるよ！
